Grove Prototype

This prototype explores AI-augmented scientific discovery in new energy materials. Below is an AI-evaluated dataset of 50 synthetic candidate materials. Each entry includes structural, physical, and heuristic properties — and composite AI scores for ranking and exploration.

Dataset Overview

Column Description
material_id Synthetic label (e.g., Mat-001 to Mat-050). Not real compounds.

Core Physical Properties

Column Description
band_gap_eV Electronic band gap in electron volts (eV).
0–1 eV → conductive
1–2 eV → semiconducting (e.g., silicon ≈ 1.1 eV)
3+ eV → insulating
critical_temp_K Superconducting critical temperature in Kelvin.
<10 K = low / not useful, 50–100 K = interesting
ionic_conductivity_S_per_cm Ionic conductivity (log scale). Higher = better electrolyte.
Range: 10⁻⁸ to 10⁻¹ S/cm
formation_energy_eV_per_atom Formation energy per atom. Negative = stable; Positive = unstable.
density_g_per_cm3 Mass density in g/cm³. Relevant for energy density & manufacturing.

Heuristic Scores

Column Description
stability_score
